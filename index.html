<count-down>

A revolutionary new way to count down to your favourite occasion

<count-down> is a custom HTML tag built using Polymer and Web Components. Using it is as simple as including a HTML tag:

<count-down hour="11" minute="20" usewords></count-down>

Of course, you'll need to include the Web Components Polyfill and import the <count-down> element using HTML imports:

<head>
    ...
    <script src="//cdnjs.cloudflare.com/ajax/libs/polymer/0.3.3/platform.js"></script>
    <link rel="import" href="//keanulee.github.io/count-down/count-down.html"> 
</head>
